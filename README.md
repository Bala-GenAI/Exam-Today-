# Industry & Academic Training Brochure

## Overview
This brochure showcases comprehensive training programs designed to bridge the gap between academic education and industry requirements.

## Features

### Training Programs Covered:
- **Core Programming**: Python Programming
- **Data Science**: Statistical analysis, ML fundamentals, data visualization
- **Artificial Intelligence**: AI, Machine Learning, Deep Learning
- **Generative AI**: LLMs, GPT models, prompt engineering
- **Agentic AI**: Autonomous systems, multi-agent frameworks
- **Langchain**: Framework for building LLM applications
- **RAG**: Retrieval-Augmented Generation systems
- **Cloud Platforms**: AWS, Azure, GCP
- **CI/CD Tools**: Jenkins, GitLab CI/CD, GitHub Actions, Docker, Kubernetes, etc.

### Project Types:
- **Mini Projects**: 1-2 weeks duration
- **Major Projects**: 4-8 weeks duration
- **Live Projects**: 8-12 weeks duration

### Academic Levels:
- Undergraduate Students
- Graduate Students (Master's)
- PhD Scholars
- Working Professionals
- Career Switchers

## How to Use

1. **View the Brochure**: Open `brochure.html` in any modern web browser
2. **Print**: Use browser's print function (Ctrl+P) for a printable version
3. **Customize**: Edit the HTML file to add your specific contact information, branding, or additional content

## Customization

To customize the brochure:
- Update contact information in the "Get in Touch" section
- Modify color schemes by editing the CSS gradient values
- Add your logo in the header section
- Update course descriptions and pricing as needed

## Progressive Web App (PWA) Features

### Install as Mobile/Web App
- **Installable**: Can be installed on phones and computers like a native app
- **Offline Support**: Works without internet connection after first visit
- **App-like Experience**: Full-screen, standalone mode
- **Fast Loading**: Cached for instant access
- **Shareable Link**: Send a link that opens the brochure directly

### How to Deploy & Share

1. **Quick Deployment Options:**
   - **GitHub Pages** (Free): Upload to GitHub and enable Pages
   - **Netlify** (Free): Drag & drop folder, get instant link
   - **Vercel** (Free): Import project, deploy in seconds
   - See `DEPLOYMENT.md` for detailed instructions

2. **Share the Link:**
   - Once deployed, share the URL with anyone
   - They can open it on any device (phone, tablet, computer)
   - They can install it as an app for quick access
   - Works offline after first visit

3. **Mobile Installation:**
   - **Android**: Tap menu → "Add to Home Screen" or "Install App" button
   - **iPhone**: Tap Share → "Add to Home Screen" or "Install App" button
   - **Desktop**: Click install icon in address bar or "Install App" button
